# Linux Cheat Sheet

## Navigation Commands

pwd - Show current directory

ls - List files and directories

cd directory_name - Change directory

## File Operations

mkdir folder_name

touch file.txt

cp source destination

mv old_name new_name

rm file.txt

## Permissions

chmod 755 file.sh

chmod +x script.sh

chown user:user file.txt

## Networking Commands

ip a

ping google.com

netstat -tulnp

traceroute google.com

## Package Management

sudo apt update

sudo apt upgrade

sudo apt install package_name

dpkg -l

## System Information

whoami

uname -a

hostname

df -h

free -h
